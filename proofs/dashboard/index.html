Divergence Parameters

Target Nₑ 2000
Genetic drift accumulates with time and shrinks with population size. Here source is anchored (F_S = 0), so only target drift changes with divergence settings.
Migration Parameters
Scaled migration rate (M) 1.00
Higher gene flow (larger M) means less differentiation between demes. At equilibrium, within-deme coalescence time is always 2 units; between-deme time grows as gene flow decreases.
Present-Day Baseline
Past influence is explicit: source signal is transported by drift scaling r = (1 - F_target) / (1 - F_source), then converted to present-day R²/AUC/Brier.
Source R² (present-day baseline) 0.50
Outcome prevalence (for Brier) 0.10
Glossary
F_ST — fraction of genetic variation due to population structure (0 = identical, 1 = fully diverged)
PGS — polygenic score: weighted sum of genetic variants predicting a trait
R²S — present-day source variance explained (baseline performance)
Nₑ — effective population size: genetic equivalent of census size
M — scaled migration: higher = more gene flow between demes
δ — equilibrium differentiation between demes under migration-drift balance
MAF — minor allele frequency: how common the less-frequent variant is
